# Hi ![](https://user-images.githubusercontent.com/18350557/176309783-0785949b-9127-417c-8b55-ab5a4333674e.gif) I am Arkadeep Das

<h3 align="center">A CSE undergraduate passionate about AI/ML and GenAI</h3>

I enjoy developing AI-powered applications that combine Large Language Models (LLMs), Retrieval-Augmented Generation (RAG) and multi-agent workflows to solve real-world problems. My work focuses on transforming AI concepts into deployable products using tools like LangChain, Streamlit, TensorFlow, CrewAI and modern LLM APIs.

Alongside development, I have a strong interest in Data Structures & Algorithms, Graph Theory, Deep Learning and software engineering fundamentals.

* 🌍  I'm based in Kolkata, India

* 🖥️  See my portfolio at [Portfolio](http://www.linkedin.com/in/arkadeep-das-65494a28a)

* ✉️  You can contact me at [https://www.linkedin.com/in/arkadeep-das-65494a28a](mailto:https://www.linkedin.com/in/arkadeep-das-65494a28a)

* 🚀  I'm currently working on [Wikipedia RAG Chatbot](http://github.com/ArkadeepDas4444/wikipedia-chatbot)

* 🧠  I'm currently learning Agentic AI systems, Advanced RAG pipelines, AI-powered developer tools and Applied Machine Learning & LLM Engineering

* 🤝  I'm open to collaborating on Innovative AI projects, Hackathons, Research opportunities and Open-source contributions
